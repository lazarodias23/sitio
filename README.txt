INSTRUÇÕES DE ORGANIZAÇÃO:

1. Crie uma pasta chamada `css` e mova o arquivo `css/style.css` para lá.
2. Crie uma pasta chamada `js` e mova os arquivos `js/script.js` e `js/chatbot.js` para lá.
3. Crie uma pasta chamada `img` para colocar suas imagens.
4. Crie uma pasta chamada `videos` para colocar seus vídeos.

Para o Chatbot funcionar:
Abra `js/chatbot.js` e insira sua chave da API Gemini na variável `API_KEY`.
